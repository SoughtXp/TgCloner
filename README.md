  ☑ Formatted text cloning<br>

## Installation

```bash
# Clone the repository
git clone https://github.com/SoughtXp/TGClonerX.git
cd TGClonerX

# Install dependencies
pip install -r requirements.txt

# Start the dashboard
python main.py
```
